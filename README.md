# PickupPal Frontend

This is a modern, responsive single-page website for **PickupPal**, built with React and Tailwind CSS. It presents the Terms of Service and Privacy Policy in a clean, human-friendly format.

## Tech Stack
- **React** (Vite)
- **Tailwind CSS**
- **Lucide React** (Icons)
- **Framer Motion** (Animations)

## Getting Started

To run this project locally, follow these steps:

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start the Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Project Structure
- `src/App.jsx`: Main application component containing the layout and sections.
- `src/LegalContent.js`: Structured content for the legal sections.
- `src/index.css`: Tailwind CSS imports and global styles.
- `tailwind.config.js`: Tailwind CSS configuration with custom brand colors.

## Features
- **Sticky Navbar**: Responsive navigation with a mobile menu.
- **Smooth Scrolling**: Anchor-linked navigation for easy reading.
- **Human-Friendly Terms**: Legal content written in plain English.
- **Important Notices**: Visually distinct warning boxes for key information.
- **Final Agreement**: Interactive agreement section with a checkbox.
- **Back to Top**: Quick navigation back to the header.
- **Animations**: Subtle scroll-reveal animations using Framer Motion.
